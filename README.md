# XT-HS2-Small-Somatic
End-to-end somatic SNV and indel detection pipeline for Agilent SureSelect XT HS2 consensus sequencing data. Designed for low-frequency variant calling in post-mortem tissue using Agilent's UMI consensus pipeline (AGeNT CReaK) and GATK Mutect2. Validated on fresh-frozen bulk brain samples from MSA cases and controls.
